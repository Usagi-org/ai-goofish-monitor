## ADDED Requirements

### Requirement: 用户可以通过卖家 ID 监控店铺商品
系统 SHALL 支持通过卖家 ID 创建监控任务，自动抓取该卖家店铺内的所有商品。

#### Scenario: 创建卖家监控任务
- **WHEN** 用户提供卖家 ID 和可选的关键词过滤
- **THEN** 系统创建卖家维度监控任务，记录卖家 ID、过滤关键词、抓取频率

#### Scenario: 抓取卖家商品列表
- **WHEN** 任务触发时
- **THEN** 系统通过闲鱼搜索接口（关键词 + 卖家 ID 过滤）获取商品列表，最多抓取 50 件商品

#### Scenario: 卖家信息持久化
- **WHEN** 首次遇到新卖家 ID 时
- **THEN** 系统在 seller_info 表中创建记录，包含卖家 ID、昵称、头像、最后更新时间

### Requirement: 卖家维度过滤规则
系统 SHALL 支持针对卖家维度的过滤规则（如屏蔽特定卖家、优先监控某卖家）。

#### Scenario: 卖家黑名单
- **WHEN** 卖家 ID 在黑名单中
- **THEN** 系统跳过该卖家的所有商品，不进行分析也不推送通知

#### Scenario: 卖家白名单优先级
- **WHEN** 卖家 ID 在白名单中
- **THEN** 系统优先抓取该卖家商品，AI 分析优先级提升

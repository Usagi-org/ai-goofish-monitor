## ADDED Requirements

### Requirement: 结果卡片显示商品想要数
系统必须在结果卡片上展示商品的"想要数"字段，帮助用户快速判断商品热度。

#### Scenario: 正常显示想要数
- **WHEN** 商品数据中包含"想要人数"字段且有值
- **THEN** 在结果卡片上显示"想要数：X"或"X 人想要"

#### Scenario: 想要数为空时的处理
- **WHEN** 商品数据中"想要人数"字段为空或不存在
- **THEN** 显示"—"或不显示该字段

#### Scenario: 大数据格式化
- **WHEN** 想要数超过 1000
- **THEN** 显示为"1k+"或"1.2k"等格式化形式
